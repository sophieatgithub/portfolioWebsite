From Windows to Linux

by Sophie Rossiter

Created July 25, 2020

My first computer, in the 90s, came with Windows preinstalled, as has every laptop I have owned since. As I delved further into the world of software, the desire grew to have more control over what my computer did, how and when it did it. So I made the leap and decided to install Ubuntu Desktop.

Ubuntu was widely recommended as any easy Linux distribution for those looking to move from Windows. The steps were quite straightforward.

  1. Create a bootable USB stick. Ubuntu has an excellent, step by step guide.
  2. Download Ubuntu onto the USB stick.
  3. Backup anything to keep. For this, I used Google Drive and a second USB stick.
  4. Turn the computer off and on again. When it first starts up interrupt the boot using the appropriate key (F1, F2, F10, F11, F12, Esc or Del).
  5. Select the USB stick from the menu.
  6. Follow the instructions on-screen to install Ubuntu.

I ran into some difficulty here; I was unable to access the internet. It turned out that I needed a wireless LAN adapter. I downloaded it using a separate laptop and installed via a USB stick. I found this answer on ask Ubuntu very helpful.

If I were to go through this process again, I would have the necessary wireless LAN adapter downloaded onto USB before starting.
